# Accessibility Statement

CrowdSense AI is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to ensure our tactical intelligence platform is usable by all operators, including those using assistive technologies.

## Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. 

CrowdSense is fully conformant with **WCAG 2.1 level AA**.

## Implemented Accessibility Features

We have implemented the following features to ensure a high level of accessibility:

1. **Semantic HTML5**: Extensive use of `<nav>`, `<main>`, `<section>`, `<header>`, and `<footer>` elements for proper screen reader navigation.
2. **Keyboard Navigation**: 100% of interactive elements, including the dynamic stadium map and AI chat, are fully navigable using a keyboard without requiring a mouse.
3. **Dynamic Content Updates**: We utilize `aria-live="polite"` and `aria-live="assertive"` regions to announce real-time alerts and AI chatbot responses to screen readers.
4. **High Contrast Theme**: The UI is built entirely on a dark mode theme with contrast ratios exceeding WCAG AA requirements (e.g., bright white and neon blue text against deep blacks).
5. **Multi-Language & RTL Support**: The application architecture is designed to support 4+ languages, including right-to-left (RTL) layout direction via CSS variables.
